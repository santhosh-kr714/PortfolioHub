# PortfolioHub

The premier 100% free, AI-powered portfolio generation platform for developers, designers, and creatives.

## Features

- **100% Free Forever**: No premium tiers, no paywalls.
- **AI-Powered**: Integrated with OpenAI, Gemini, and Claude for automatic bio writing, project descriptions, and chatbots.
- **Custom Domains**: Connect your own domain instantly.
- **Template Library**: Switch between Awwwards-winning templates (Glassmorphism, Cyberpunk, Minimalist) with one click.
- **Plugin Ecosystem**: Add GitHub activity, Spotify listening status, and Calendly booking widgets.
- **Analytics Engine**: Track unique visitors, resume downloads, and traffic sources.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion, Zustand
- **Backend**: FastAPI (Python), SQLAlchemy, SQLite (Dev) / PostgreSQL (Prod)
- **Deployment**: Docker, Docker Compose, GitHub Actions CI/CD
